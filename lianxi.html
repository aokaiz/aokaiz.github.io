塑料成型设备练习题

判断题(1')

  1. 机械式振动筛的是利用凸轮装置产生振动的。( )
  2. 圆筒筛的结构特点是筛选效率高、卫生性差、筛网的维修方便。 ( )
  3. 振动筛通常适合筛选粒状树脂和密度较大的填料。 ( )
  4. 塑料混合机除具有均匀混合的效果以外，对物料还具有一定的预塑化作用，有利于后序加工。( )
  5. 在塑料制品成型中，对物料进行筛选、过滤的目的之一是防止金属杂质混入物料中。 ( )
  6. 在成型之前，对物料进行预热可提高物料的混合塑化效率，缩短成型周期。 ( )
  7. 辐射预热干燥不适用于大批量物料的预热干燥。 ( )
  8. 远红外线预热干燥的特点是：设备规模小、制造简便、成本低、易于推广。 ( )
  9. 三辊研磨机是较常用的一种浆料研磨设备。 ( )
  10. 在三辊研磨机中，各辊筒是相向而转的。 ( )
  11. 真空干燥不适用于干燥在加热时易氧化变色的物料。 ( )
  12. 平动筛不适合筛选粉状物料。 ( )
  13. 对原料预热干燥的形式有热风预热干燥、真空干燥和辐射预热干燥。 ( )
  14. 料斗式塑料干燥机主要用于挤塑成型或注塑成型直接加料的过程。 ( )
  15. 塑料混合机混合的时间越长越利于物料混合均匀。 ( )
  16. 塑料混合机的混合机理主要是塑化作用。 ( )
  17. 塑料混合机主要通过离心率、循环、旋转等作用来混合物料。 ( )
  18. 搅拌装置主要由搅拌桨和主轴传动部分组成，通常大搅拌浆采用较慢的搅拌速度。 ( )
  19. 在开炼机中，调距转轮顺时针旋转时辊筒间距减小，反之增大。 ( )
  20. 连续密炼机具有很好的自洁性。 ( )
  21. 挤出成型过程是一种间断式生产过程，所以生产效率相对较低。 ( )
  22. 挤塑成型设备是三大成型设备之一，通过它来完成注塑产品的生产。 ( )
  23. 挤出生产线中的挤出模具是挤出辅机的组成部分之一。 ( )
  24. 螺杆长径比是指螺杆有效工作长度与螺杆半径D的比值。 ( )
  25. 物料在挤出过程中热量的来源主要有两个，一个是机筒外部加热器提供的热量，另一个是物料与机筒、螺杆之间的剪切和物料与物料之间所产生的摩擦所产生的热量。 ( )
  26. 挤出机的螺杆直径对挤出产量影响较小。 ( )
  27. 随螺槽深度的增加，挤出机的产量会相应增大。 ( )
  28. 挤出机主机的结构和组成随制品类型的变化而变化。 ( )
  29. 目前挤塑机最常见的形式是中小型卧式单螺杆非排气整体式普通成型挤塑机，简称普通挤塑机。( )
  30. 挤出机的挤压系统主要由螺杆、机筒、连接体及喷嘴组成。 ( )
  31. 在挤出设备中，若用孔径不同的两层过滤网，应将孔径较细的网紧贴分流板。 ( )
  32. 尖形的螺杆头利于加工高黏度、热稳定性差的物料。 ( )
  33. 双螺杆主要由挤压系统、传动系统、加热冷却系统、排气系统、加料系统和电气控制系统等系统组成。 ( )
  34. 双螺杆挤出机可分为啮合双螺杆和非啮合双螺杆，其中，啮合型适合做产品，而非啮合型适合做材料的改性。 ( )
  35. 在双螺杆挤出机组成的挤出生产线中，冷却辅机如果是喷淋式的，其冷却效率要比浸浴式低。 ( )
  36. 固体输送理论假设螺槽是矩形的且深度不变。( )
  37. 横流是熔体在螺杆均化段发生的流动类型之一，又称为环流，通常只考虑它对制品质量的影响而忽略对产量的影响。 ( )
  38. 在实际生产中用封闭挤出来清洗机筒或更换物料。 ( )
  39. 一根好的螺杆，在保证塑化质量的前提下，名义比功率应尽可能高。 ( )
  40. 对于结晶性大、熔融温度范围窄的物料通常熔融段长度L2设计得短一些。( )
  41. 分离型螺杆中的BM螺杆，副螺棱与主螺棱的相交始于加料段末，终于熔融段初。 ( )
  42. 向内异向旋转双螺杆的“压延效应”比向外异向旋转更严重。( )
  43. 对于要求制品透明度高的薄膜生产通常采用平挤下吹法。 ( )
  44. 注射机通常由注射系统、合模系统、液压传动系统和电气控制系统组成。 ( )
  45. 注塑系统主要由塑化装置、料斗、计量装置、螺杆传动装置、注塑油缸、注塑座整体移动油缸组成。 ( )
  46. 注塑系统主要由开合模装置、顶出装置、安全保护装置等组成。 ( )
  47. 螺杆式注塑系统是目前应用最广泛的注塑系统。 ( )
  48. 在注塑过程中射台前进通常在合模前完成。 ( )
  49. 注塑机参数主要包括理论注塑容积、注塑压力、注塑速率、塑化能力等。 ( )
  50. 注射速率是指在注射时单位时间内从喷嘴射出的熔料体积流率。 ( )
  51. 注塑机只能用于热塑性塑料的注射成型。 ( )
  52. 注塑机的锁模力一定要大于注塑压力。 ( )
  53. 注塑机锁模时，一定要达到最大锁模力。 ( )
  54. 为了防止熔体“流延”，低发泡注塑机选用锁闭式注嘴，并通过降低背压抑制发泡剂分解 ( )
  55. 注塑机的动模板不属于注塑系统。 ( )
  56. 全液压注塑机的开合模行程与模具的厚度没有关系。 ( )
  57. 液控锁闭式注嘴主要用于加工低黏度的物料。 ( )
  58. 一般对于中低粘度的物料，注塑机螺杆要用止逆型螺杆头。 ( )
  59. 电动注塑机具有高精度、低碳、低排放的特点。 ( )
  60. 机筒的壁厚越厚越利于节能减排，能在一定程度上提高厂家利润。 ( )
  61. 柱塞式注射机的缺点是塑化均一性差、压力损失大、工艺条件不稳定。 ( )
  62. 注塑机是一种生产效率高，易于自动化生产的设备。 ( )
  63. 注射成型可用于热塑性塑料的成型，也可用于热固性塑料的成型，但不能用于橡胶。 ( )
  64. 合模装置是为保证成型模具可靠闭紧、实现模具启闭动作及顶出制品的机械部件。 ( )
  65. PVC塑料注射成型时通常选用的螺杆头部结构形式为分流型。 ( )
  66. 开合模速度是反应注塑机工作效率的重要参数，会直接影响到成型周期的长短。 ( )
  67. 热固性塑料注射成型时，物料在料筒中的停留时间的控制幅度可以大于热塑性塑料。 ( )
  68. 压延成型设备由供料系统、压延成型机、压延辅机、加热冷却系统以及电气控制系统五大部分组成。 ( )
